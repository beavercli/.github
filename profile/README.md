# BeaverCLI

Building a unified documentation and knowledge search experience for your GitHub ecosystem.

## What we are building
- CLI that indexes and searches docs across multiple repos without leaving your terminal.
- Fast remote search tuned for developer knowledge sharing.
- Opinionated workflows that integrate with existing tooling.

## Projects
- `beaver_api` — core API and indexing service.
- CLI client — upcoming tool for querying your org-wide knowledge base.
